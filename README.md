# Hotel-Booking-Demand-Analysis
Hotel Booking Demand Analysis & Cancellation Prediction

This project analyses hotel booking data to identify key business insights related to customer behaviour, booking demand, pricing strategies, and cancellation patterns. Using Python and machine learning techniques, the project transforms raw booking data into actionable recommendations that can support revenue optimisation and operational decision-making within the hospitality industry.

The project follows a complete data science workflow:

Data Collection and Understanding
Data Cleaning and Preparation
Exploratory Data Analysis (EDA)
Data Visualisation
Customer Segmentation
Machine Learning Model Development
Model Evaluation
Business Recommendations

# Business Problem

Hotels frequently experience booking cancellations that negatively affect revenue, occupancy planning, and resource allocation. The objective of this project is to analyse historical booking data and develop a machine learning solution capable of identifying booking patterns and predicting the likelihood of cancellation.

# 📌 Business Value

This project helps hospitality organisations:

Reduce revenue loss from cancellations
Improve occupancy forecasting
Optimise pricing strategies
Improve customer targeting
Support data-driven decision making
